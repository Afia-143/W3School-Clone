HTML Links - Hyperlinks

HTML links are hyperlinks. You can click on a link and jump to another document.

When you move the mouse over a link, the mouse arrow will turn into a little hand.

HTML Links - Syntax:

The HTML <a> tag defines a hyperlink. It has the following syntax:

<a href="url" > link text </a>

<a href="https://www.w3schools.com/"> Visit W3Schools.com! </a>

Absolute URLs vs Relative URLs:

The example above is using an absolute URL (a full web address) in the href attribute.

A local link (a link to a page within the same website) is specified with a relative URL

(without the "https://www" part):

<h2> Absolute URLs </h2>
<p> <a href="https://www.w3.org/"> W3C </a> </p>
<p> <a href="https://www.google.com/"> Google </a> </p>

<h2> Relative URLs </h2>
<p> <a href="html_images.asp"> HTML Images </a> </p>
<p> <a href="/css/default.asp"> CSS Tutorial </a> </p>

You can also make an Image as a link, an Email as a link and

also a Button, about which you can learn through the given youtube tutorial video.



HTML Images

Images can improve the design and the appearance of a web page.

You can also add images in your website as per your choice.

HTML Images - Syntax:

The HTML <img> tag is used to embed an image in a web page.

The <img> tag is empty, it contains attributes only, and does not have a closing tag.

The <img> tag has two required attributes:

  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image if respective browser is unable to find image

<img src="url" alt="alternatetext">

Absolute Images vs Relative Images:

Some web sites point to an image on another server.

To point to an image on another server, you must specify an absolute (full) URL in the src attribute.

If you have your images in a sub-folder, you must include the folder name

in the src attribute, that would be the realtive URL.

<h2> Absolute URLs </h2>
<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

<h2> Relative URLs </h2>
<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">
